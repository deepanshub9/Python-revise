# Python Revise

This repository contains a collection of simple Python projects and exercises created for practice and learning purposes. All scripts are beginner-friendly and focus on core Python programming concepts.

## What's inside?

Below are the main Python scripts included in this repo, with easy-to-understand descriptions:

### Projects & Scripts

- **Calculator_with_history.py**  
  A simple calculator that supports addition, subtraction, multiplication, and division. It keeps a history of your calculations and lets you view or clear it.

- **Digital_Clock.py**  
  A graphical digital clock using Tkinter that displays the current time.

- **Fake_News_headline_generator.py**  
  This fun script randomly generates "fake news" headlines by mixing up subjects, actions, and places.

- **Gussing_game.py**  
  (Password guessing game) Pick a difficulty (easy, medium, hard) and try to guess the secret word. It provides hints and tracks your attempts.

- **Ordermenu.py**  
  A menu ordering system for a fictional restaurant. Enter items to order, see your order summary and total bill.

- **Rent_Calculator.py**  
  Calculate individual rent by entering house rent, food expenses, electricity units, cost per unit, and number of people.

- **Rock_Paper_Scissor.py**  
  Classic Rock-Paper-Scissors game where you play against the computer.

- **guessing_number.py**  
  Guess a secret number between 1 and 10. You have 3 chances to find it.

- **math_operation.py**  
  Demonstrates basic math operations (add, subtract, multiply, divide) by calling functions from `practice_python.py`.

- **practice_python.py**  
  Contains several Python practice functions and code snippets, including basic arithmetic functions.

- **student_Grade_System.py**  
  Manage a small system for students' grades: add, update, delete, and display student grades.

### Other files

- **history.txt**  
  This is used to store calculator history.

## How to use

Just download or clone this repository and run any of the `.py` scripts using Python. For GUI-related code, make sure you have Tkinter installed (usually included by default).

```bash
# Example: run the calculator
python Calculator_with_history.py
```

Feel free to review and try out the programs for learning and practice!
